# Red Hat Learning Path Data for Developer Hub

This set of learning paths advertise our official learning journeys.

## Usage

Follow the [Red Hat Developer Hub documentation](https://docs.redhat.com/en/documentation/red_hat_developer_hub/1.7/html/customizing_red_hat_developer_hub/proc-customize-rhdh-learning-paths_about-software-catalogs#proc-customizing-the-learning-paths-by-using-a-hosted-json-file_about-software-catalogs) to add these to your internal developer portal.
